تحديث V22 الشامل:

1) تفويض التأشيرات
- إصلاح خطأ Warning / undefined variable في قائمة المكتب المخلص.
- إبقاء شكل خانة البحث كما هو مع جلب بيانات المكاتب فعلياً.
- إضافة معاينة/طباعة داخل نفس الصفحة.
- تحسين تعديل/إلغاء التفويض وإبقاء العدد غير قابل للتعديل.

2) دليل المعاملات
- إضافة بحث عام كبير أعلى الصفحة.
- إضافة فلاتر لحظية متعددة الاختيار: الخدمة، الحالة، العميل، المورد، الوكيل، المكتب المخلص، نوع العميل، المتبقي، العملة.
- ربط الحالات بالخدمات: عند اختيار خدمات محددة تظهر الحالات المرتبطة بها فقط.
- إضافة معاينة داخل نفس الصفحة مع A4 وطباعة/Excel/PDF.
- تحسين تصميم الجداول مع خطوط وصفوف وأعمدة وتمرير أفقي.
- إضافة إشعارات داخل النظام للتعديل/المردود/التسليم.

3) حالة المعاملات
- إضافة صفحة جديدة باسم حالة المعاملات.
- فلاتر لحظية مثل ترحيل المعاملات.
- تحديد معاملات وتغيير حالتها جماعياً مع منع تغيير حالات خدمات مختلفة.
- مربع داخلي لتأكيد التغيير واختيار الحالة.

4) العملاء والتسليم
- إضافة خيار تسليم في اختصارات الثلاث نقاط.
- إضافة زر تسليم داخل ملف العميل.
- مربع تسليم داخل الصفحة مع تنبيه إذا على العميل متبقي.
- طباعة سند التسليم من نفس المربع.
- تحسين تنسيق السعر والعملة داخل مربع إضافة خدمة.

5) المحاسبة
- تحسين ميزان المراجعة: من/إلى، اختيار عملة أو الكل، نتيجة الفرق صفر/غير متوازن، ومعاينة A4.

- إضافة صفحة حركة النشاط التجاري.
- تصحيح ربط الموردين والمكاتب المخلصة بالحسابات المناسبة.
- نقل المورد الرئيسي إلى مكانه تحت الموردين وليس كفرع رئيسي مستقل.
