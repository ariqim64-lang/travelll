V33 - تقارير محاسبية عالمية

تمت إعادة بناء صفحتي ميزان المراجعة والأرباح والخسائر بدون دمج العملات.

ميزان المراجعة:
- فلاتر من/إلى، عملات متعددة، نوع الحساب، مستوى الحساب، إظهار الحسابات، بحث عام.
- كل عملة تظهر كجدول مستقل.
- افتتاحي مدين/دائن، حركة الفترة، ختامي مدين/دائن، عدد القيود، آخر حركة.
- إجمالي وفارق لكل عملة.
- إعدادات أعمدة ومعاينة قبل الطباعة.
- الضغط على الحساب يفتح كشف حسابه لنفس الفترة والعملة.

الأرباح والخسائر:
- كل عملة جدول مستقل بدون خلط.
- إيرادات، مصروفات/تكاليف، صافي ربح/خسارة، النسبة، عدد القيود.
- إجمالي وصافي لكل عملة أسفل جدولها.
- تقرير كتابي لكل عملة.
- إعدادات أعمدة ومعاينة وطباعة/Excel/PDF.

الفحص:
- PHP syntax
- JavaScript syntax
- ZIP integrity
