V31 - السبب الحقيقي لعدم اختفاء/ظهور الفلاتر بشكل صحيح

بعد الفحص سطر بسطر للنسخة، السبب الفعلي كان في CSS وليس في JavaScript فقط:

في style.css توجد قواعد قديمة مثل:
.multi-drop label { display:block!important; }

هذا يعني أن JavaScript عندما يعمل:
label.style.display = 'none'
فإن الإخفاء لا ينجح لأن CSS القديم فيه !important يجبر كل خيارات القوائم على الظهور.

لذلك كانت:
- ملاحظة "لا توجد حالات لهذه المعاملات" تظهر حتى عند وجود حالات.
- خيارات الحالات التي يفترض أن تختفي لا تختفي.
- الربط بين الخدمة والحالة يبدو وكأنه لا يعمل.

الإصلاح في V31:
- لم أعد أعتمد على style.display فقط.
- أضفت classes خاصة مثل v31-force-hidden و v31-visible.
- أضفت CSS أقوى بـ !important يخفي الخيارات فعلياً.
- أضفت filters_v31.js ويتم تحميله بعد app.js.
- رفعت الكاش إلى v=310.
