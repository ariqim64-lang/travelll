V17 - إصلاح فعلي لاختصارات العملاء وترحيل المعاملات

ما تم إصلاحه:
1) سبب عدم ظهور المربعات في صفحة العملاء:
- تم تحويل اختصارات الثلاث نقاط إلى أزرار ثابتة بربط JavaScript مباشر وآمن بدلاً من روابط/hover قد تُغلق أو لا تنفذ داخل الجدول.
- تم رفع طبقة المربعات فوق كل الصفحة وضبط z-index والـ overflow حتى لا تختفي خلف الجدول أو تُقص.
- تم جعل قائمة الثلاث نقاط تفتح بالضغط وتبقى ثابتة حتى يختار المستخدم.

2) صفحة العملاء:
- إضافة خدمة تفتح مربع داخل نفس الصفحة.
- مكتب مخلص / تغيير الحالة / مردود / عرض أو إنزال التأشيرة تفتح مربعات داخل نفس الصفحة.
- إذا العميل لديه أكثر من معاملة تظهر قائمة اختيار المعاملة أولاً.
- الإشعارات داخلية وليست alert المتصفح.

3) ترحيل المعاملات:
- اختيار المكتب المخلص أصبح خانة بحث صغيرة داخل نفس الخانة.
- فلاتر المكاتب والخدمات صارت قوائم منسدلة صغيرة مع مربعات اختيار وليس مربعات كبيرة ظاهرة.
- تم تصغير القوائم وتحديد ارتفاعها حتى لا تخرب شكل الصفحة.
- البحث والفرز يعملان مباشرة.

4) الفحص:
- PHP syntax: سليم.
- JavaScript syntax: سليم.
- ZIP integrity: سليم.
