# مراجعة V50 الخامسة - الصفحات والإشعارات وحركة النشاط التجاري

تمت هذه المراجعة على نسخة V49 مع التركيز على:
- فتح الصفحات وعدم تعطل صفحة بسبب صفحة أخرى.
- صفحة الإشعارات وصلاحياتها.
- صفحة التقارير وسجل النشاط والأجهزة.
- صفحة حركة النشاط التجاري.
- عزل الفروع والمحاسبة متعددة العملات.

## أهم الإصلاحات

1. صفحة الإشعارات
- تم منع ظهور نموذج إنشاء الإشعار وقائمة المستخدمين لأي مستخدم لا يملك صلاحية إدارة الإشعارات.
- بقيت قراءة الإشعارات الخاصة بالمستخدم متاحة له فقط.
- بقيت الإدارة قادرة على عرض وفرز إشعارات كل المستخدمين حسب الصلاحية.
- تم تثبيت إعدادات لون الإشعار حتى تحفظ وتظهر بشكل صحيح.

2. حركة النشاط التجاري
- تم منع خلط العملات عند حساب الربح والخسارة.
- تكلفة المورد وتكلفة المكتب المخلص يتم تحويلها إلى عملة بيع المعاملة قبل حساب ربح المعاملة.
- تم استبعاد الترحيلات الملغاة من تكلفة المكتب المخلص.
- يتم عرض دخل العميل والتكاليف والربح بعملة المعاملة نفسها حتى تكون نتيجة كل معاملة مفهومة ومحاسبياً أقرب للصحة.

3. سجل النشاط والأجهزة
- المدير أو صاحب صلاحية عرض سجل النشاط يرى كل السجلات.
- المستخدم العادي يرى سجلاته وأجهزته فقط.
- تم حماية تعليم سجل النشاط كمقروء بحيث لا يستطيع المستخدم تعليم سجل غيره كمقروء.

4. الأداء وقوة البنية
- أضيفت فهارس أداء آمنة تلقائياً عند التشغيل للجداول الثقيلة:
  ledger_entries, transactions, office_transfers, notifications, activity_logs, login_device_logs.
- الفهارس تنشأ فقط إذا كانت غير موجودة، ولا توقف النظام إذا فشل إنشاء فهرس.

## الفحوصات المنفذة
- فحص PHP Syntax لكل ملفات PHP.
- فحص JavaScript Syntax لكل ملفات JS.
- فحص عدم وجود دوال PHP مكررة.
- فحص أن كل روابط route لها case في النظام.
- فحص أن ضغط ZIP النهائي سليم.

## ملاحظة مهمة
