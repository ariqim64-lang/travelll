تحديث V13

أهم التعديلات:
1) تغيير اسم صفحة حالة التأشيرات إلى: تفويض التأشيرات.
2) في تفويض التأشيرات:
   - إضافة فلتر: الكل / المفوضة / غير المفوضة.
   - ربط مربع التحديد مع خانة العدد.
   - عند وضع الصح يتم إدراج كامل غير المفوض تلقائياً.
   - عند كتابة عدد يتم وضع الصح تلقائياً.
   - رقم التفويض أصبح إجبارياً عند التفويض، والتنبيه يظهر داخل النظام.
   - إضافة قسم إلغاء التفويض مع تأكيد داخلي.
3) في ترحيل المعاملات:
   - إضافة فلتر: الكل / المرحلة / غير المرحلة.
   - إضافة قسم إلغاء الترحيل مع تأكيد داخلي.
   - إضافة عمود is_reversed للترحيلات.
4) تطبيق تنسيق الجداول بخطوط رأسية رفيعة ومقابض توسيع على:
   - الخدمات والحالات.
   - المهن.
   - البنوك.
   - الصناديق.
   - الموردين.
   - الوكلاء.
   - المكاتب المخلصة.
   - الموظفين.
   - المخزون.
   - تفويض التأشيرات.
   - ترحيل المعاملات.
5) إضافة عرض/تعديل/حفظ/رجوع لصفحات الخدمات والمهن والجهات والموظفين.
6) منع تكرار السجلات الأساسية مع إشعار داخلي أنيق.
7) إضافة شجرة صلاحيات تفصيلية في صفحة المستخدمون والصلاحيات.

ملاحظة:
إذا كانت قاعدة البيانات موجودة مسبقاً، سيتم محاولة إضافة العمود الجديد is_reversed تلقائياً عبر ensure_accounting_defaults، كما أضيف في install.sql.
